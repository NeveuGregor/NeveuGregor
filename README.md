### 🛠️ Selected work

- **🔐 [mcp-noctua](https://github.com/NeveuGregor/mcp-noctua)** — MCP server
  exposing a *sandboxed* pentest toolbox (sqlmap, nuclei, ffuf, httpx, naabu,
  katana…) to an LLM orchestrator, for authorized security audits. Agentic
  security, offensive side.
- **🔌 [mcp-obsidian](https://github.com/NeveuGregor/mcp-obsidian)** —
  filesystem-based MCP server for an Obsidian vault: read/search/write notes
  directly on disk, no REST plugin required.
- **🔌 [mcp-redmineApi](https://github.com/NeveuGregor/mcp-redmineApi)** — MCP
  server driving a Redmine instance from Claude via its REST API (issues, time
  logging, metadata, allow-listed uploads).
- **📐 [claude-md-playbook](https://github.com/NeveuGregor/claude-md-playbook)** —
  a field-tested playbook for writing `CLAUDE.md` files: treat them as
  infrastructure — triggered rules, testable claims, converged through use (EN + FR).
- **🧠 [claude-obsidian-system](https://github.com/NeveuGregor/claude-obsidian-system)** —
  a personal knowledge architecture: an Obsidian vault as a compounding second
  brain (LLM-Wiki + Zettelkasten + Evergreen) wired to Claude Code skills, MCP
  and persistent memory.
- **🛠️ [cc-switcher](https://github.com/NeveuGregor/cc-switcher)** — switch
  Claude Code authentication between multiple accounts (personal / work) with a
  single shared `~/.claude`.
- **🧰 [DevRewind](https://github.com/NeveuGregor/DevRewind)** — local, single-user
  activity journal to prep annual reviews: PHP/SQLite REST API (prepared
  statements, FTS5 search, PHPUnit-tested) + Vue 3 / TypeScript frontend.
- **🧰 DevReview** — full-stack code-review companion app (PHP · SQLite · Vue 3).
  *Going public soon.*

> Flagship work lives inside private/legacy repos: a role-separated sub-agent
> ecosystem (reviewer, security sentinel, devil's advocate, pre-mortem) and a
> Semgrep security skill bundle, both operating on a ~50,000-file PHP codebase.
